# bumeok-jjikmeok
> **2025 카카오 관광데이터 활용 공모전 출품작**  
> *맛집은 먹고, 추억은 찍고! 부산 여행을 완성하는 '부먹찍먹'*
<br></br>

## 🎯 프로젝트 개요
<div align="center">
  <img width="1920" height="1080" alt="부먹찍먹" src="https://github.com/user-attachments/assets/2f48ad81-e431-4809-bad5-e8f383954407" />

부먹찍먹은 **부산을 여행하는 관광객**과 **지역의 로컬 경험을 즐기고 싶은 현지인**을 위한 모바일 플랫폼으로, **분산된 관광지/맛집 정보를 한 눈에 보고, 취향에 맞는 여행 루트를 쉽게 찾기 어려운 문제를 해결**합니다.</br>
사용자 맞춤형 테마 추천, 퍼즐맵 기반 미션, 지도 기반 탐색, 후기 및 추천 커뮤니티와 배지 시스템을 통해 재미있고 직관적인 여행 경험을 제공하며, Tour API를 활용해 실시간으로 신뢰성 높은 부산 관광지와 맛집 정보를 제공합니다.</br>
이를 통해 사용자는 성취감을 느끼며 재방문을 유도하고, 신뢰도 높은 정보와 현지인 추천을 기반으로 풍부한 부산 여행 경험을 얻을 수 있게 도와줍니다.</br>

</br></br>

## 👩🏻‍💻 Backend Tech Stack
| 구분            | 기술/도구                             |
| ------------- | --------------------------------- |
| **언어**        | Java 17                           |
| **프레임워크**     | Spring Boot 3, Spring Security    |
| **데이터베이스**    | MySQL 8                           |
| **JPA / ORM** | Spring Data JPA, Hibernate        |
| **빌드 도구**     | Gradle                            |
| **서버**        | AWS EC2 (Ubuntu)                  |
| **스토리지**      | AWS S3 (이미지 업로드/삭제)               |
| **인증**        | JWT 기반 인증 / 인가 처리                 |
| **배포**        | GitHub Actions + Nginx            |
| **API 문서**    | Swagger (springdoc-openapi-ui 사용), Notion |
